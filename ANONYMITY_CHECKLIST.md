# Anonymity Checklist

- No article PDF, article source file, or author metadata is included.
- Personal local paths were removed from the GitHub artifact.
- API keys and private tokens are not included.
- Python caches, LaTeX logs, old ZIP packages, and full episode logs are not included.
- Trained model checkpoint files are not included in the GitHub artifact.
- The package is scoped to Paper A only and does not include other projects.

Known non-identifying terms:

- The phrase `LM Studio` refers to the local LLM serving application.
- Model names such as `Gemma`, `Qwen`, `Nemotron`, and `LFM2.5` are public model identifiers.
